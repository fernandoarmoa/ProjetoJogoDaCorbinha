Jogo da Cobrinha

Como Jogar

Objetivo:

Pegar o pontinho vermelho sem tocar no corpo da cobrinha.

Instruções:

  • Seta Direita: para mover para a direita
  • Seta Esquerda: para mover para a esquerda
  • Seta Cima: para mover para cima
  • Seta Baixo: para mover para baixo
